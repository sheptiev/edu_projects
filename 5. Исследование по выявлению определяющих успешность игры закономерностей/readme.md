# Исследование закономерностей, определяющих успешность игры 
Учебная задача: Нужно провести исследование, позволяющее выявить определяющие успешность игры закономерности. Это позволит в дальнейшем 
сделать ставку на потенциально популярный продукт и спланировать рекламные кампании.

Из открытых источников доступны исторические данные о продажах игр, оценки пользователей и экспертов, жанры и платформы (например, Xbox или PlayStation).
Данные представлены до 2016 года (включительно). Нужно отработать принцип работы с данными, чтобы его можно было использовать в дальнейшем (например, в 2026 году).

Исследование состоит из следующих этапов:
<ol>
  <li>Открытие файла с данными и изучение общей информации</li>
  <li>Подготовка данных</li>
  <li>Проведение исследовательского анализа данных</li>
  <li>Составление портрета пользователя каждого региона</li>
  <li>Проверка гипотез</li>
  <li>Общий вывод</li>
</ol>
