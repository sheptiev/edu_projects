Цель учебного проекта: исследовать товарный ассортимент интернет-магазина товаров для дома «Пока все ещё тут». Дополнительно выделить основной и дополнительный ассортимент.
Заказчиком исследования по условию задания выступает менеджер, отвечающий за актуальность товарного ассортимента.

Исследование состоит из следующих этапов:<br>
<ul>
    <li>Изучить общую информацию о данных из датасета</li>
    <li>Преподготовка данных</li>
    <li>Исследовательский анализ данных</li>
    <li>Формулирование и проверка статистических гипотез</li>
    <li>Общие выводы, рекомендации. Подготовка итоговой презентации</li>
</ul>
<br>
Каждый из этапов сопровождается комментариями, а в конце сделан общий вывод с рекомендациями в принятии решений.
