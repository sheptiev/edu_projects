# Исследование вакансий в Краснодаре, представленных на портале Работа России
Цель работы: исследовать рынок вакансий Краснодара, представленный на портале Работа России (trudvsem.ru).<br>
Задачи:<ul><li>Исследовать уровень предлагаемой заработной платы;</li><li>Изучить распределение вакансий по сферам;</li><li>Выявить различные особенности текущего рынка вакансий.</li></ul>

Работа состоит из следующих этапов
<ol>
  <li>Загрузка и предобработка данных</li>
  <li>Исследовательский анализ</li>
  <li>Общий вывод по работе</li>
  <li>Выгрузка исследования для построения дашборда</li>
</ol>

В исследовании построены проведён анализ количества вакансий по сферам, типам занятости, названиям вакансий, требуемого опыта. Также проведён анализ средней предлагаемой заработной платы в различных разрезах.
<br>
По итогу работы построен дашборд, доступный по ссылке: <a href='https://public.tableau.com/app/profile/vasily3456/viz/DashbordWork/sheet4'> ссылка на дашборд </a>
